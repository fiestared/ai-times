AIツール — 何に使うか・いくらか（公式ページと照合）

主要なAIツールの用途と料金（月額）を、各社の公式価格ページと照合した一覧です。掲載価格は各ツールの公式ページに実在する数字だけを載せています（curlで生取得して確認）。裏取りできない価格は載せません（ChatGPT・Gemini・Perplexity・Midjourney などは公式ページがcurlを拒否/JS描画のため、確認でき次第 追加します＝fail closed）。$は米ドルです。
